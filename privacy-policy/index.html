Politique de Confidentialité

Dernière mise à jour : 24 Février 2026

1. Introduction

Votre vie privée est notre priorité. Cette politique décrit comment l'application « Mama & Bébé », éditée par KSA MEDIA LLC, collecte, utilise et protège vos données personnelles.

2. Données Collectées

  • Informations de Compte : Adresse email (uniquement pour l'authentification via Firebase Auth).
  • Données de Santé : Informations liées à la grossesse (semaine de grossesse, poids, rendez-vous médicaux) que vous saisissez manuellement. Ces données sont stockées de manière sécurisée sur Google Firebase.
  • Données Techniques : Identifiants techniques anonymes pour assurer le bon fonctionnement de l'application et détecter les erreurs (via Sentry).

3. Utilisation des Données

Vos données sont utilisées exclusivement pour :

  • Vous fournir les fonctionnalités de suivi de grossesse.
  • Synchroniser vos données entre vos appareils (si vous êtes connectée).
  • Améliorer la stabilité de l'application (détection anonyme de crashs via Sentry).

Nous ne vendons, ne louons et ne partageons aucune donnée personnelle avec des tiers à des fins commerciales ou publicitaires.

4. Services Tiers

Nous utilisons les services sécurisés suivants :

  • Google Firebase : Hébergement sécurisé des données et authentification. Politique Firebase.
  • Sentry : Détection anonyme des erreurs techniques. Aucune donnée personnelle n'est transmise. Politique Sentry.
  • Notifications Push (Expo / Firebase Cloud Messaging) : Un identifiant anonyme d'appareil ("push token") est envoyé aux serveurs d'Apple (APNs) ou de Google (FCM) uniquement pour délivrer les rappels que vous avez activés. Cet identifiant ne permet pas de vous identifier personnellement.
  • Google Firebase Analytics : Données d'utilisation anonymes et agrégées (nombre d'ouvertures, fonctionnalités utilisées). Aucune donnée personnelle transmise. Politique Google.

5. Sécurité

Toutes les communications entre l'application et nos serveurs sont chiffrées via HTTPS/TLS. Vos données sont stockées sur l'infrastructure sécurisée de Google Cloud Platform (Firebase).

6. Vos Droits

Vous disposez à tout moment du droit d'accès, de rectification et de suppression de vos données :

  • Suppression dans l'app : Allez dans Profil → Paramètres → Supprimer mon compte. Toutes vos données seront immédiatement et définitivement effacées.
  • Par email : En nous contactant à contact@mama-et-bebe.com pour toute demande relative à vos données.
